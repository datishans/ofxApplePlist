# ofxApplePlist

openFrameworks addon for reading Apple Property list files (plists).

## Dependencies

- openFrameworks 0.9.8
- Uses Poco (::Any)
- Runs on macOS (should work / untested on Linux & Windows since plists are not used outside macOS).
